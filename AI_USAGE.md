
## AI Usage Statement

During the development of this project, AI tools (such as Gemini/ChatGPT) were utilized strictly as productivity assistants, not as core developers. Specifically, AI was used for:

* **Environment Troubleshooting:** Resolving OS-specific workflow issues (e.g., handling Git/Docker file permission differences between Windows CMD and Ubuntu WSL).
* **Boilerplate Generation:** Writing the boilerplate code for `generate_data.py` to quickly create the 500,000-row mock dataset without wasting development time.
* **Documentation:** Proofreading and structuring this README file and presentation notes for better readability.

The core ETL pipeline logic, DuckDB queries, zero-copy Pandas integration, and architectural benchmarking decisions were researched and implemented entirely by the author.
